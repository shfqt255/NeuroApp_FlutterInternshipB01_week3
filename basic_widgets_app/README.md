A simple **Flutter Profile Card UI** project that demonstrates how to build a clean user interface using core Flutter widgets. 

---

## Overview

This application displays a profile card containing:
- A profile image
- Name and profession
- Social media icons

The UI is built using **StatelessWidget** and standard **Material Design** components.

---

## Features

- Clean and minimal profile card layout
- Uses `Scaffold` and `AppBar`
- Styled `Container` with:
  - Rounded corners
  - Box shadow
- Circular profile image using `CircleAvatar`
- Vertical layout using `Column`
- Social icons displayed using a separate reusable widget
- Beginner-friendly and well-commented code
