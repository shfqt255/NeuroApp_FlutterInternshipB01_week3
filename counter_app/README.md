# Flutter Counter App

This app allows the user to **increment**, **decrement**, and **reset** a counter value.

---

## Features

- Stateful widget implementation
- Counter state management using `setState()`
- Increment counter value
- Decrement counter value
- Reset counter to zero
- Clean and simple UI using Material Design
